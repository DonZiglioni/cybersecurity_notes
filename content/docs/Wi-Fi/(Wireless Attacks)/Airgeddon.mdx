## Quickstart

```bash
git clone --depth 1 https://github.com/v1s1t0r1sh3r3/airgeddon.git
cd airgeddon
sudo bash airgeddon.sh
```

## Usage

```bash
sudo airgeddon
```
 - Put interface into monitor mode
 - option 5: Handshake Tools Menu
 - option 4: Explore for targets
 - The data column shows that someone or something is attached to the network
 - "ctrl c" to stop and select target network
 - option 5: Capture handshake
 - option 1: Deauth / ....
 - If option 1 fails, run the next option and so on
 - Press Y for yes once it captures the handshake
 - Save file and crack!
